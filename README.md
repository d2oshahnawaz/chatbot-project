# 🤖 Voice Chatbot Project

A Python-based intelligent voice chatbot that can interact with users through text and speech. This project integrates Natural Language Processing (NLP), Machine Learning, and a graphical user interface (GUI) to create an interactive chatbot system.

---

## 🚀 Features

* 🎤 Voice-based interaction (Speech Recognition)
* 💬 Text-based chatbot interface
* 🧠 Machine Learning model for response generation
* 📊 Custom dataset support (CSV)
* 🖥️ GUI-based chatbot interface
* 🔊 Text-to-Speech response system

---

## 📁 Project Structure

```
├── app.py                  # Main application entry point
├── chatbot_gui.py         # GUI interface for chatbot
├── chatbot_interface.py   # Chat logic handler
├── chatbot_model.py       # ML model definition
├── train_model.py         # Model training script
├── voice_chatbot.py       # Voice interaction module
├── dataset.csv            # Training dataset
└── README.md              # Project documentation
```

---

## ⚙️ Installation

### Step 1: Clone the repository

```
git clone https://github.com/d2oshahnawaz/chatbot-project.git
cd chatbot-project
```

### Step 2: Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Usage

### Run the chatbot application

```
python app.py
```

### Train the model (optional)

```
python train_model.py
```
